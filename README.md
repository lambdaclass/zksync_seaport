# zksync_seaport
Seaport implementation on zkSync Era. Seaport is a marketplace protocol for safely and efficiently buying and selling NFTs.

## Running `era-test-node`

### Setup

Clone the `era-test-node` repo.

```
make setup

```

### Update

Updates `era-test-node`.

```
make update
```

### Run

Runs `era-test-node`.

```
make run
```
